
# Developing emlearn-motion

For those that wish to hack on emlearn-motion itself.

Contribution guidelines can be found in [CONTRIBUTING.md](CONTRIBUTING.md).

If you just wish to use it as a library, see instead the [usage guide](https://emlearn-motion.readthedocs.io/en/latest/user_guide.html).


## Running tests

```
FIXME: document
```

## Building documentation

Make sure to have dev dependencies

```
pip install -r requirements.dev.txt
```

Run the Sphinx build

```
make -C docs/ html
```

Open the frontpage in browser `docs/_build/html/index.html`
